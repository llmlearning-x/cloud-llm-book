## 阿里云核心产品清单

### 模型服务
- **DashScope 灵积**：统一模型 API 调用平台
- **通义千问 Qwen**：阿里云自研大模型系列
- **通义万相 Wanx**：图像生成模型
- **通义听悟**：语音转写与分析

### 平台服务
- **PAI**：机器学习平台（DSW/EAS/Lab）
- **函数计算 FC**：Serverless 计算
- **容器服务 ACK/ASK**：Kubernetes 容器编排
- **API 网关**：API 管理与调用

### 数据服务
- **OSS**：对象存储
- **Elasticsearch**：向量检索
- **RDS/PolarDB**：关系数据库
