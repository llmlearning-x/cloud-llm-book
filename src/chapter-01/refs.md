## 关键概念澄清

### MaaS (Model as a Service)
定义：通过云服务提供大模型能力的模式，包括模型训练、微调、推理 API 等全生命周期管理。

核心特征：
- 按需付费，降低使用门槛
- 弹性伸缩，应对流量波动
- 托管运维，专注业务逻辑

### 大模型应用开发挑战
1. **成本**: Token 计费模式下如何优化使用量
2. **延迟**: 实时应用场景下的响应时间要求
3. **安全**: 数据隐私、内容合规、防注入攻击
4. **可观测性**: 调试困难、问题定位复杂
