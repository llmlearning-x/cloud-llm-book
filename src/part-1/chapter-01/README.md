## 1.1 人工智能概述

### 1.1.1 从图灵测试到 ChatGPT

人工智能（Artificial Intelligence，AI）是计算机科学的一个分支，旨在创造能够模拟人类智能的机器。自 1956 年达特茅斯会议正式提出 AI 概念以来，这个领域经历了多次高潮与低谷。

```
时间线：
1956 - AI 概念诞生
1980s - 专家系统兴起
1997 - 深蓝击败国际象棋冠军
2012 - AlexNet 开启深度学习时代
2016 - AlphaGo 击败围棋冠军
2022 - ChatGPT 引爆大模型时代
2023 - GPT-4、LLaMA、通义千问等百模大战
```

### 1.1.2 什么是大语言模型？

大语言模型（Large Language Model，LLM）是一类基于深度学习技术、使用大规模文本数据训练的神经网络模型。它们能够理解和生成人类语言，完成翻译、写作、对话等各种语言任务。

**LLM 的核心特征：**

| 特征 | 说明 |
|------|------|
| 规模巨大 | 参数量从数十亿到数千亿不等 |
| 预训练 | 在海量文本上学习通用语言能力 |
| 生成式 | 能够创造性地产出文本内容 |
| 涌现能力 | 规模增大后出现意想不到的能力 |

### 1.1.3 主流大模型概览

| 模型 | 开发商 | 特点 |
|------|--------|------|

## 1.2 云计算基础

### 1.2.1 云计算的三种服务模式

```
┌─────────────────────────────────────────────────────────┐
│                      SaaS (软件即服务)                   │
│              直接使用的应用（钉钉、飞书、企业微信）          │
├─────────────────────────────────────────────────────────┤
│                      PaaS (平台即服务)                   │
│            开发平台（函数计算、数据库、消息队列）            │
├─────────────────────────────────────────────────────────┤
│                      IaaS (基础设施即服务)                │
│              虚拟化资源（云服务器、存储、网络）              │
└─────────────────────────────────────────────────────────┘
```

### 1.2.2 为什么 AI 需要云计算？

1. **算力需求**：训练大模型需要海量 GPU 算力
2. **弹性扩展**：推理服务需要根据负载自动扩缩
3. **成本优化**：按需付费，无需自建机房
4. **快速迭代**：云上环境便于实验和部署

### 1.2.3 阿里云 AI 产品体系

阿里云提供了完整的大模型应用开发产品矩阵：

```
DashScope 灵积（模型服务）
    ├── 通义千问 Qwen（对话）
    ├── 通义万相 Wanx（图像）
    ├── 通义听悟（语音）
    └── Embedding（向量）

PAI（机器学习平台）
    ├── DSW（数据科学环境）
    ├── EAS（模型部署服务）
    └── 算法市场

基础服务
    ├── 函数计算 FC（Serverless）
    ├── 容器服务 ACK（Kubernetes）
    ├── 对象存储 OSS
    └── API 网关
```
